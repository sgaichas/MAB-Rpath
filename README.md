# Mass balance Rpath model for Mid-Atlantic Bight in the Northeast US Large Marine Ecosystem

To generate balanced models used in cross system comparisons, run `MAB_Rpath_no_disc.R` script

To run ecosense algorithm and generate model ensembles, run `Sarah_R/MAB_sense.R`

Balanced models are in `outputs` folder

Models used in ecosense are `MAB_params_Rpath_no_disc.RData` and `MAB_Rpath_no_disc.RData`
